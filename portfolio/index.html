Bolt

- A fast paced bow cobat game!

Bolt is a bow combat game in minecraft where players can play a range of different games modes including Capture the Flag, Freeze Tag, Free for All, Capture the Point and Team Death Match. The map contains a wide range of maps from small to big; to open to closed. Coving a wide range of play styles. The map itself also utilizes a custom kill handling system to bypass the restrictions of the default one.

Nicknames Ultimate

Nicknames Ultimate is a spigot plugin that adds in the ability for players to nickname themselves in game. It is a highly inclusive plugin with support for changing of their chat name, tab and supports placeholders from all your favourite places. It also has support for banned words; colour and format codes and you can even hide that you are nicknamed.

Honey Missile wars

Honey Missile Wars is an update of the classic Missile wars map (with permission from Sethbing/ Cubehamster) where we have taken the classic concept and expanded on it. This includes a range of new missiles, new arenas and whole new gamemods born from these options. In support of this we also added all the old missiles so you can still play the classic version.

Gamemode 4

- No Mods Just Modules

Gamemode 4 is a collection of datapacks that change and modify the game play experience. I helped add multiple modules over time and help update and bug fix even more. As part of Gamemode 4 there is a public server that contains most of the available modules in an open and welcoming environment to allow people to play together. As part of this server I am both a moderator and one of the core system admins in which I have created a range of plugins for and supporting datapacks.

Security Guard 2

- Midnight Hiest

Security Guard 2 is a remake of the old map Security Guard. There is 2 aims in this game depending on your team. The thief’s needs to steel as many relics as they possibly can before the Guards capture them. The Guards as said before need to try and capture all the thief’s and save some relics!
This game includes a range of mechanics including CCTV cameras, Escape potions, AI and an escape mode where once the time is up your objective becomes to escape the museum!

Git Datapack Manager

Git Datapack Manager is a spigot plugin that integrates with GitHub to allow large teams to work on datapacks in a single location and commit live updates on the server. This plugin is designed so that teams who are working on a project can easily make use of GitHub team tools to work together and update the server without everyone needing individual file access.

GFD Bot

GFD bot is the discord bot that we are using in the GFD discord. It is mainly written in typescript using discord.js as a base. We are using the bot to handle our weekly gamenights and other non-regular gamenights we do. The bot is designed to interface with the servers I am running that we use to automate them were possible. It also handles all our notification roles and moderation needs!

PBE

PBE is a Minecraft dungeon crawler made using datapacks. It was designed to be a completive game which is completed in small groups of up to 4 in a race to the end. It would have had a global scoreboard on a website. As part of the project we have a 100% custom head up display, this included custom health, armour, stamina and magic displays. As part of the system I was responsible for and make sure it felt dynamic and updated when expected. As part of the HUD I was also working on implementing a custom attack cool down display.
