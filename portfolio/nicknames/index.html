Nicknames Ultimate Plugin

The Plugin

Nicknames Ultimate is the ultimate nickname plugins containing a range of features surrounding changing your nickname!
This Nicknames plugin adds support for changing your name in every location including tab, chat and even your own player tag (coming soon). As well as all of this it also adds the support to toggle your nicknames prefix (if you have permission) so staff have the ability to hide!
On top of this I have also added a range of checks to prevent a wide ranges of issues including:

  • Fuzzy name matching. Ensure no one is portending to be someone else!
  • Fuzzy blocked word matching. Ensure no one is trying to use banned words in their nickname!
  • Fuzzy nickname matching. Ensure no one is trying to get the same nick as someone else, even when they are logged off!
  • Colour code checking. Ensure no one that is not meant to is using colour codes!
  • Format code checking. Ensure no one that is not meant to is using formatting codes!
  • Join checking. Did someone have a nickname that is no longer valid? Thatâ€™s ok when they join it will confirm that their nickname is still valid (when the option is enabled).



Other Features

Within the plugin there is a system to support multiple diffrent storage, currently setup for MySQL and local json storage!
The plugin also contains commands to look up who is currently using what nicknames. So if you not sure who someone is, just check with /realname [username]
There is then also a wide range of admin commands to reset plays nicknames, change them and even hide if you or someone else is nicknamed!

Highly Configurable!

The whole plugin is configurable, from who can do what with permissions, to the messages sent in response to commands. This means that you can controll which features are enabled or disabled, which words are blocked and even if nicknames will be saved or checked on login!
